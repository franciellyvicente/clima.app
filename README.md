🌤️ Clima App

Aplicação web para consulta de clima em tempo real por nome de cidade.

Funcionalidades


Busca de clima por nome da cidade
Exibe temperatura atual, máxima e mínima
Exibe umidade, velocidade do vento e descrição do tempo
Ícone do clima atualizado em tempo real


Tecnologias


HTML, CSS e JavaScript
OpenWeatherMap API


Como usar


Clone o repositório:


bash   git clone https://github.com/franciellyvicente/clima.app.git


Abra o arquivo index.html no navegador.
Digite o nome de uma cidade e pressione Enter.


Aviso

Este projeto utiliza uma API key pública. Não compartilhe nem exponha sua chave em repositórios públicos.
