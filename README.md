# 🌤️ Clima App

Aplicação web para consulta de clima em tempo real, com busca por nome de cidade. Projeto simples e funcional, focado em consumo de API e exibição de dados meteorológicos de forma clara.

## 🛠️ Tecnologias

- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## 🚀 Funcionalidades

-  Busca de clima por nome da cidade
-  Exibe temperatura atual, máxima e mínima
-  Exibe umidade e velocidade do vento
-  Descrição do tempo com ícone atualizado em tempo real

## 🎨 Design

- Paleta de cores em tons de azul e cinza
- Tipografia: Poppins
- Ícone do clima animado, alternando conforme a condição (sol, chuva, nuvens)

## ▶️ Como usar

1. Clone o repositório:
```bash
git clone https://github.com/franciellyvicente/clima.app.git
```
2. Abra o arquivo `index.html` no navegador.
3. Digite o nome de uma cidade e pressione Enter.

## ⚠️ Aviso

Este projeto utiliza uma API key pública. Não compartilhe nem exponha sua chave em repositórios públicos.
